📌 最重要的三条原则（划重点）
  1. 战斗指令只能在群里用，私聊只能给 DM 录怪物。私聊使用 .攻击 这种指令会被拒绝
  2. 指令开头 . 或 。 都可以（海豹两个都认，赞美海豹！）
  3. @ 某人时，用 QQ 的 @ 功能真的戳他一下，不要手写他的名字（不会真的有人手输吧？）
🚀 你是新人？从这三步起学习如何使用
  1. 如果你会施法：.法槽 法师 5（把"法师 5"换成你的职业和等级）
  2. 装上武器：.装备 长剑
  3. （可选）想双手握武器：.持握
① 开局设置（法槽 · 装备 · 持握）
.法槽 .sslots
它是干嘛的：告诉系统你是什么职业、几级，系统帮你算好有几个几环法术位。施法者第一件事就做这个。非施法者也建议做（能顺便算好戏法威力）。
怎么输入指令（职业和等级要一对一对写）：
  • .法槽 法师 5 我是5级法师
  • .法槽 圣武士 3 诗人 2 兼职：3级圣武士+2级诗人，有什么就写什么
  • .法槽 邪术师 5 邪术师用的是"契约法术位"，在指令上会有所不同
  • .法槽 show 查看我现在还剩几个法术位
  • .法槽 reset 把法术位全部清零（一般用不到）
  • .法槽 调整 3 +1 手动给3环法术位+1（撤回失败时补回用）
  • .法槽 调整 契约 -1 契约法术位也支持加减
支持哪些职业名（照抄，不要写错哦）：
全施法法师 / 术士 / 牧师 / 吟游诗人（或写"诗人"）/ 德鲁伊
半施法圣武士 / 游侠
三分之一奥法骑士 / 诡术师
特殊邪术师（或写"契约"）—— 契约法术位，大休/小休恢复
输入完会看到什么：
骰子会回复"法术位已设置（施法等级X）：1环:4 2环:3…"这样一行行列出来。
可能遇到的问题：
请成对输入你只写了职业没写等级（或反过来）。正确是"职业 数字 职业 数字"这样成对写。
等级需为1-20等级填了0、超过20、或不是数字。
法术位不对职业名拼错了（比如"法师师"），重新输入一遍。
.装备 .sequip
它是干嘛的：设定你手里拿的武器。之后 .攻击 就用这把武器算伤害。
怎么输入指令：
  • .装备 长剑 最简单，装一把长剑
  • .装备 长剑 双手 双手握（可调武器双手伤害更高）
  • .装备 长剑 +1 +1魔法武器
  • .装备 长剑 双手 +2 双手握的+2魔法武器
配套查询指令：
  • .武器 list 看有哪些武器可以装
  • .武器 info 长剑 看长剑的详细数据
  • .武器 看我现在装的是什么
可能遇到的问题：
未知武器武器名写错了。先 .武器 list 看正确名字照着抄。
伤害偏低可能该双手握没握。用 .持握 切一下。
.持握 .sgrip
它是干嘛的：在单手和双手之间切换。只对"可调武器"（长剑、战锤这种）有效，双手握伤害骰更大。
怎么输入指令：
  • .持握 直接写，不用加任何东西，每写一次该指令都切换一次单手/双手
可能遇到的问题：
非可调武器你的武器不能调握法（比如匕首、巨剑）。忽略即可，本来就固定。
② 攻击（这里指的是使用武器攻击）
.攻击 .sattack
它是干嘛的：用你装备的武器攻击目标，系统自动算命中没命中、暴击、伤害、还有怪物的抗性/免疫。
最简单的指令输入方法：
  • .攻击 哥布林 攻击DM放出来的怪，直接写怪名
  • .攻击 @张三 攻击队友，用QQ的@功能戳他
加选项（可以任意组合、顺序随便）：
  • .攻击 哥布林 优势 优势=掷两个d20取高的那个
  • .攻击 哥布林 劣势 劣势=掷两个d20取低的那个
  • .攻击 哥布林 命中+1d4 手动让这次攻击命中临时加1d4（用于没来得及挂效果、或只用一次的情况；祝福术这种持续buff请用 .效果 挂上去，会自动生效）
  • .攻击 哥布林 伤害+2d6 手动让伤害临时加2d6（同上；猎人印记、偷袭等持续伤害请用 .效果，效果指令的都会自动生效，不需要每次手动加）
  • .攻击 哥布林 附赠 标记这是附赠动作（只是显示，不影响计算，实际也没有任何实际用处，仅是让别人知道你是用附赠打）
  • .攻击 哥布林 优势 伤害+1d6 全部叠一起也行
群体攻击（一次攻击多个目标）：
  • .攻击 @张三 @李四 @王五 @多个玩家=群攻玩家
  • .攻击 哥布林1 哥布林2 哥布林3 写多个怪名=群攻怪物（空格隔开）
  • .攻击 哥布林1 哥布林2 优势 选项写最后，作用于每次攻击
注意：玩家和怪物不能在同一条指令里混着攻击，要么全@玩家，要么全写怪名。
特殊：
  • .攻击 self 攻击自己（也可写 自己 / 我），但是真的要打自己吗......？
可能遇到的问题：
请先.装备武器还没装武器。先 .装备 长剑。
尚未启用模组DM还没把怪放出来。让DM先 .怪物 启用 模组名。
未找到怪物怪名写错，或DM没录这只。让DM发 .怪物 list 看正确怪名。
命中-x看起来比较奇怪你开了巨武器大师/神射手（命中-5换伤害+10），由于脚本会内置自动计算，所以是正常现象。感到疑惑的话可以先自己手算一下真实命中，以及关闭专长指令是： .专长 关闭 巨武器大师。（示例）
③ 施法与反应
.施法 .scast
它是干嘛的：放法术。系统从法术库查这个法术，自动扣法术位、算命中或让目标豁免、算伤害。
怎么输入指令：
  • .施法 火球术 哥布林 对怪物放火球
  • .施法 魔法飞弹 @张三 对玩家放
  • .施法 疗伤术 @李四 治疗队友
  • .施法 疗伤术 治疗/增益类不写目标，默认给自己
  • .施法 火球术 哥布林 升环1 用高一环的法术位放，伤害更高
  • .施法 灼热射线 哥布林 优势 攻击类法术可以带优势
  • .施法 火球术 哥布林 难度+2 临时提高目标要豁免的DC
  • .施法 灼热射线 哥布林 激励 用上诗人给的激励骰（见后面激励说明）
可能遇到的问题：
法术库未找到法术名写错了，用全名。骰子会提示相似名字，对照一下看看是不是你要找的。
法术位不足这环用完了。.法槽 show 看剩余，或 .大休 恢复。
优势已忽略火球术这种是"目标做豁免"不是你掷命中，优势对它没用。正常提示，不用管。
优势没生效优势挂在你自己身上，确认buff给对人了（用 .效果 列表 查）。
.反应 .reaction · 银光锐语
它是干嘛的：消耗1个1环法术位，给敌人的下次攻击/豁免加"劣势"，同时给友方（可以是你自己）下次攻击/豁免加"优势"。
怎么输入指令：
  • .反应 银光锐语 @敌人 @友方 敌友都是玩家
  • .反应 银光锐语 哥布林 @友方 敌人是怪物就直接写怪名
  • .反应 银光锐语 @敌人 不写友方，默认优势给自己
可能遇到的问题：
法术位不足没有1环法术位了。
请指定敌方你既没@人也没写怪名。
优势挂错人记住顺序：第一个是敌人，第二个是友方。敌人是怪物时直接写怪名，那个@就是友方。
④ 增益 · 效果系统（新手也能看懂，不准看不懂/(ㄒoㄒ)/~~）
.效果 .effect
它是干嘛的：给某人身上加一个临时状态（buff/debuff），能模拟绝大多数"加骰子"或"给优劣势"的东西，比如祝福术、猎人印记、各种诗人歌等。
添加效果的完整格式：
  • .效果 添加 @张三 祝福 1d4 attack,save auto
这一长串每个位置是什么意思，往下看逐字拆解 👇
添加操作：添加一个效果
@张三加给谁（不@就是加给自己）
祝福效果名，自己起名字随便写
1d4加多少骰。纯优劣势不加骰的话这里填 0
attack,save作用范围（见下方【什么是作用范围】）
auto模式（见下方【什么是模式】）
❓ 什么是"作用范围"？
就是"这个效果在你做哪种动作时生效"。举例：祝福术让你命中和豁免都+1d4，那作用范围就写 attack,save（命中和豁免）；猎人印记只加伤害，就写 damage。多个范围用英文逗号连起来，不留空格。
attack你的攻击命中时生效（武器和法术攻击都算）
damage你的武器伤害生效
save你做豁免检定时生效
spellAttack只在法术攻击命中时生效
spellDamage只在法术伤害时生效
all上面全部场景都生效（懒人选项）
❓ 什么是"模式"？
就是"这个效果怎么触发、会不会用一次就没"。auto 是最省心的：挂上去后每次自动加，直到你手动删。manual "用的时候才触发"——攻击时要在指令里额外写上效果名它才生效，（激励就是这种）。advantage/disadvantage 是直接给优势/劣势，触发一次就自动消失。
auto自动生效，每次都加，除非手动删，否则会一直挂着。适合：祝福术、猎人印记这种持续buff
manual要在攻击时写上效果名才触发。适合：想控制什么时候用的一次性资源
advantage给一次优势，用完自动消失
disadvantage给一次劣势，用完自动消失
consume加在最后，表示"用一次就消失"，一般配 manual 用
照抄就能用的实例：
  • .效果 添加 @张三 祝福 1d4 attack,save auto 祝福术：命中和豁免都+1d4，持续
  • .效果 添加 @张三 猎人印记 1d6 damage auto 伤害+1d6，持续
  • .效果 添加 @张三 战斗律动 1d6 attack manual 手动触发：攻击时需要输入"战斗律动"才会触发
  • .效果 添加 @张三 精准 0 attack advantage 给一次攻击优势
查看 / 删除：
  • .效果 列表 @张三 看张三身上有哪些效果
  • .效果 列表 看自己身上的
  • .效果 删除 @张三 祝福 删掉某一个效果
  • .效果 清除 @张三 清空这个人全部效果
可能遇到的问题：
效果没生效作用范围写错了。想加命中就要写 attack，写成 damage 只会加伤害。
名字/骰值错位@之后要紧跟效果名，中间别多输入空格或奇怪字符。
出现莫名+0身上有之前误加的脏效果。.效果 清除 清干净再重加。
⑤ 专长 · 能力 · 激励 · 休息
.激励 .inspire 诗人专用
它是干嘛的：诗人给某人一个激励骰。被激励的人下次攻击/施法/豁免时，输入"激励"两个字就能加这个骰子。
怎么输入指令（前提是需要录入诗人！）：
  • .激励 @张三 骰子大小按你的诗人等级自动定
  • .激励 @张三 1d10 手动指定骰子大小
  • .激励 @自己 激励自己
被激励的人怎么用：
  • .攻击 哥布林 激励 攻击时加激励骰，用完自动消失
  • .施法 火球术 哥布林 激励
骰子怎么自动定的：
1-4级1d6
5-9级1d8
10-14级1d10
15级以上1d12
前提：诗人要先用 .法槽 诗人 X 录过等级，系统才知道你诗人几级。
可能遇到的问题：
为什么激励骰一直是1d8诗人没录职业等级。发 .法槽 诗人 X，或直接手动 .激励 @人 1d6。
明明写了激励但没加骰你身上没激励效果，让诗人先 .激励 @你。
.专长 .turn
它是干嘛的：开启有真实计算效果的专长。目前只有下面3个真的会影响数值，别的专长请看 .能力。
怎么输入指令：
  • .专长 开启 巨武器大师 近战重武器：命中-5、伤害+10
  • .专长 开启 神射手 远程武器：命中-5、伤害+10
  • .专长 开启 魔法抗性 对法术豁免有优势
  • .专长 关闭 巨武器大师 关掉
可能遇到的问题：
仅支持有效果的你开的专长（比如警觉）这套系统算不了。想记录用 .能力，想加骰用 .效果。
.能力 .ability
⚠️ 注意：这个指令只是"记事本"，不会产生任何战斗效果！用来标记"我有这个能力"给自己和DM看，实际效果DM口头处理。
怎么输入指令：
  • .能力 开启 警觉 记录一下我有警觉专长
  • .能力 关闭 盾牌大师
为什么存在：DND非常非常多的专长和特殊能力（比如魔能祈唤和超魔法），大部分（包括不限于先攻、位移、隐匿类的）这套系统都做不了自动化，但是可以给你个地方记着（也许之后会做，不过我个人更倾向使用海豹原生指令）。
休息 · 等级
  • .大休 HP回满 + 全部法术位恢复（为什么用大休不用长休，因为这个指令被海豹原生占了）
  • .小休 邪术师契约法术位自动补满，其他职业不恢复资源（为什么用小休，原因同上）
  • .等级 5 设角色总等级（会影响戏法威力）
⚠️ 如果你已经用过 .法槽 职业 等级，等级会自动算好，不用再单独用 .等级这个指令。这个指令主要给不施法的pc用（只是用来告诉非施法者可以用简写，并没有什么实际用处。。）
⑥ 撤回（如果上一条指令错了想要反悔）
.撤回 .undo
它是干嘛的：撤销上一次攻击/施法/怪攻造成的所有变化——HP恢复、法术位退回、加的buff移除。手滑写错指令就用它。
怎么输入指令：
  • .撤回 直接输入就行，后面不用加任何东西
限制：
  • 只能撤最近一次，不能连撤两次
  • 只有 45秒内有效（这个时间可以修改，具体见骰主篇，我这里默认是45秒）
  • 骰子重启后撤回记录会清空
可能遇到的问题：
撤回超时超过45秒了。去"遇到问题"页看【撤回超时但已扣了】自己改回来。
没有可撤回记录还没做过行动，或已经撤过一次了。
🎬 DM 从零开团，五步走
  1. （私聊机器人）.怪物 模组 新建 第一章副本 —— 建一个存怪物的"文件夹"
  2. （私聊）.怪物 哥布林 15 7 —— 录一只哥布林（名字 护甲值 血量）
  3. （私聊）.怪物 act 哥布林 弯刀 4 1d6+2 挥砍 —— 给它一个攻击动作
  4. （进群，群主发）.怪物 bind @DM —— 告诉系统谁是DM
  5. （群里，DM发）.怪物 启用 第一章副本 —— 把怪放到本群
做完这五步，玩家就能 .攻击 哥布林 了。
DM 录入怪物
.怪物 .npc · 录入
建议在私聊里录（不会刷屏，也有较好的隐私性）。录完在群里启用就能用。
录基础属性：
  • .怪物 哥布林 15 7 格式=名字 AC(护甲) HP(血量)
给怪物加攻击动作：
  • .怪物 act 哥布林 弯刀 4 1d6+2 挥砍
act表示"加一个动作"
哥布林给哪只怪加
弯刀动作名字（自己起）
4命中加值（怪物的攻击调整值）
1d6+2伤害骰
挥砍伤害类型
加"要豁免"的动作（如喷火）：
  • .怪物 act 火龙 火焰吐息 0 8d6 火焰 豁免:敏捷 难度:15 效果:灼烧
后面三个可选：豁免:属性（要豁免哪个属性）难度:DC（豁免难度）效果:文字（失败后的状态描述）
加反应（怪被近战攻击时自动反击）：
  • .怪物 react 卫兵 借机攻击 5 1d8+3 穿刺 格式同act
加伤害特性（免疫/抗性/易伤）：
  • .怪物 tag 火元素 免疫 火焰 火元素免疫火焰伤害
  • .怪物 tag 骷髅 抗性 钝击 骷髅对钝击减半
  • .怪物 tag 稻草人 易伤 火焰 稻草人受到火焰翻倍
模组管理（把怪分组，像不同存档）
  • .怪物 模组 新建 哥布林巢穴 建并切换到这个模组
  • .怪物 模组 切换 亡灵地窖 切到另一个
  • .怪物 模组 列表 看我有哪些模组
  • .怪物 模组 删除 哥布林巢穴 删掉整个模组
查看与删除
  • .怪物 list 列出当前模组所有怪
  • .怪物 info 哥布林 看哥布林的详细数据
  • .怪物 哥布林 clr 删掉这一只
  • .怪物 清空 清空当前模组所有怪
直接改怪物数值 .怪物 set
什么时候用：战斗中血算错了、撤回超时了、需要手动调怪物状态时。不会重置满血上限，也不需要重录动作。
  • .怪物 set 哥布林 hp 20 直接把血设为20
  • .怪物 set 哥布林 hp -5 扣5血
  • .怪物 set 哥布林 hp +8 回8血（不超过血量上限）
  • .怪物 set 哥布林 ac 15 改护甲值
  • .怪物 set 哥布林 maxhp 30 改血量上限
⚠️ 作用于「当前编辑的模组」，如果有多个模组记得先 .怪物 模组 切换 模组名 切到正在战斗的那个。

  • .怪物 启用 哥布林巢穴 DM用：把模组放到本群

  • .怪攻 火龙 火焰吐息 @张三 @李四 @王五 群体喷火，多人一起豁免
  • .怪攻 哥布林 弯刀 @张三 优势

DM 常见疑问（换DM / 多团）
Q：私聊录的怪，群里能用吗？
能。怪物按"你的账号+模组名"保存，跟在哪录无关。私聊录好，群里 .怪物 启用 模组名 就能调出来。
Q：换DM了怎么办？
.怪物 bind @新DM 会自动清空旧的模组启用状态，新DM重新 .怪物 启用 自己的模组即可。
Q：同一个群两个DM各带一团？
靠切换。轮到A带团：.怪物 bind @A + A启用自己模组；换B时重新bind+启用。同一时刻只有一个模组生效。
🔧 手动修正数值（撤回超时/需要精调）
改玩家 HP / 法术位（用海豹原生 .st）
什么时候用：撤回超过45秒撤不了了，或者需要精确改某个数值时。
  • .st show 先看看当前所有数值
  • .st hp=30 直接把HP设成30
  • .st hp+10 HP加10
  • .st hp-5 HP减5
  • .st 1环法术位=4 改1环法术位（直接设值）
  • .法槽 调整 3 +1 3环法术位加1（推荐，不用算现在剩几个）
  • .st 契约法术位=2 改契约位
属性名要写对：hp / hpmax / 1环法术位~9环法术位 / 契约法术位 / 契约法术位上限 / 契约法术环阶
改怪物 HP / AC（用 .怪物 set）
怪物存在插件里，.st 改不了它。用 .怪物 set 直接精确修改，不会重置满血上限。由DM操作。
  • .怪物 set 哥布林 hp 20 直接把血设成20
  • .怪物 set 哥布林 hp -5 扣5血
  • .怪物 set 哥布林 hp +8 回8血（不超过血量上限）
  • .怪物 set 哥布林 ac 15 改护甲值
  • .怪物 set 哥布林 maxhp 30 改血量上限
改之前确认在正确的模组（.怪物 模组 切换 模组名）。
❓ 报错自助排查（对着骰子的提示找同款）
请先 .装备 武器
为什么：你还没装备任何武器就想攻击。
发 .装备 长剑。不知道有啥武器就先 .武器 list 看列表。
该指令仅支持群聊环境
为什么：你在私聊里用了战斗指令，战斗指令只能在群里使用。
回到群里再发。私聊只能给DM录怪物。
本群尚未启用模组
为什么：DM还没把怪物放到本群。
让DM发 .怪物 启用 模组名。DM要是也不知道有啥模组，先 .怪物 模组 列表 看。
未找到怪物【xxx】
为什么：怪物名写错了，或者DM根本没录这只。
让DM发 .怪物 list 看正确怪名，一字不差照抄（包括空格）。
法术位不足
为什么：你要放的这一环法术位用完了。
先 .法槽 show 看剩多少。真没了就 .大休 恢复。
撤回超时了，但法术位 / HP 已经扣了
为什么：超过45秒撤不了，但那次行动的消耗是真实发生的。
手动补回：法术位用 .法槽 调整 3 +1（把消耗的那一环+1，数字换成实际消耗的环）；玩家HP用 .st hp+X 加回来；怪物HP用 .怪物 set 怪名 hp +X（由DM操作）。
法术库中未找到【xxx】
为什么：法术名字写错了。
骰子会提示相似的名字，照着改。要用全名，比如"火球术"不能只写"火球"。
命中出现 -2 / -x 这种奇怪数字
为什么：你开了巨武器大师或神射手专长（命中-5 换 伤害+10）。
这是正常现象，插件直接读卡算好了。不想要就 .专长 关闭 巨武器大师。还是疑惑就自己手���对照一下。
豁免类法术优势已忽略
为什么：火球术这类是"目标做豁免"，不是你掷命中，所以你的优势对它没用。
这是正常提示，不用管。优势/劣势只对"攻击骰类"法术（如灼热射线）有用。
给的优势/激励没生效
为什么：优势和激励是挂在"某个具体的人"身上的，可能挂错人了。
用 .效果 列表 @那个人 确认buff在不在他身上。激励骰还得被上了激励的人在攻击时手动输入"激励"二字才触发。
效果里出现莫名其妙的 +0
为什么：身上有之前误加的、加值是0的脏效果。
发 .效果 清除（清自己）或 .效果 清除 @那个人，然后重新加干净的效果。
法术位显示不对
为什么：职业名拼错了，或者你用的是 .等级 而不是 .法槽。
重新 .法槽 职业 等级，职业名照玩家指南的列表写。设法术位一定要用 .法槽，不是 .等级。
@了人但系统没反应 / 认错人
为什么：可能是手输了名字，或者@的顺序不对。
一定要用QQ的@功能真的戳一下。银光锐语记住：第一个是敌人，第二个是友方。
你不是本群绑定的 DM
为什么：你想用DM指令但系统不认你是DM。
让群主/骰主发 .怪物 bind @你。
只有群主或骰主可以绑定 DM
为什么：绑定DM需要群主或骰主权限，你没有。
找群主帮忙发 .怪物 bind @DM。

🛠️ 可调参数（脚本顶部 CONST 区）
UNDO_TIMEOUT撤回有效时长，默认45000毫秒=45秒，可自行改大改小
MONSTER_SAVE_BASE怪物豁免加值基数，默认15，数越大怪物豁免越低
MAX_DICE_ROLLS防刷骰上限，一般不用动

① 开局设置（属性 · 法槽 · 装备 · 角色面板）
设置角色属性（海豹原生 .st 命令）v2 推荐
它是干嘛的：把六维属性、HP、AC写进角色卡。之后攻击/施法时插件直接自动读取，不用每次手动填加值。这是 v2.0.0 的核心改进：设一次，之后不用管。
一次性设置（照着改数字复制过去）：
  • .st 力量=16 敏捷=14 体质=15 智力=10 感知=13 魅力=8
  • .st hp=42 hpmax=42 ac=16 熟练=3
⚠️ hp 和 hpmax 都要设，不然 .hp 显示不出血量条。ac 不设默认10。熟练不设则从等级自动推算。
之后可以单独改某个值：
  • .st hp=35   .st ac=18
  • .st show 查看所有当前属性值
.法槽 .sslots
它是干嘛的：告诉系统你是什么职业、几级，系统帮你算好有几个几环法术位。施法者必做。非施法者也��议做（会顺带设好角色等级，影响戏法威力）。
怎么输入（职业和等级一对一对写）：
  • .法槽 法师 5
  • .法槽 圣武士 3 诗人 2 多职业成对写
  • .法槽 邪术师 5 邪术师用"契约法术位"，短休恢复
  • .法槽 show 查看现在还剩几个法术位
  • .法槽 reset 清零
  • .法槽 调整 3 +1 手动给3环法术位+1（撤回超时时用）
  • .法槽 调整 契约 -1
支持的职业名（照抄，不要写错）：
全施法法师 / 术士 / 牧师 / 吟游诗人（或"诗人"）/ 德鲁伊
半施法圣武士 / 游侠
三分之一奥法骑士 / 诡术师
特殊邪术师（或"契约"）—— 契约法术位
.装备 .sequip
它是干嘛的：设定你手里拿的武器，之后 .攻击 就用这把算伤害。插件内置40+种武器。
  • .装备 长剑
  • .装备 长剑 双手
  • .装备 长剑 +1 +1魔法武器，命中和伤害都+1
  • .装备 长剑 双手 +2
  • .武器 list 查看所有可用武器（40+种）
  • .武器 info 长剑 查看某武器详细数据
  • .武器 查看当前装备的武器
.持握 .sgrip
它是干嘛的：切换可调武器（长剑/战锤等）的单手/双手握法。双手握伤害骰更大。
  • .持握 直接输，每写一次切换一次
.角色 / .dndshow 新
它是干嘛的：一键查看角色概要面板——HP血量条、护甲、属性调整值、当前武器、法术位、状态效果、职业资源。
  • .角色 或 .dndshow
  • .角色 @张三 查看别人
② 攻击（最常用）
.攻击 .sattack
它是干嘛的：用装备的武器攻击目标。系统自动从角色卡读取属性加值，自动算命中/暴击/伤害/怪物特性（免疫/抗性/易伤）。
最简单的指令：
  • .攻击 哥布林 攻击DM放出的怪，写怪名
  • .攻击 @张三 攻击玩家，用QQ的@戳他
加选项（任意组合，顺序随便）：
  • .攻击 哥布林 优势 掷两个d20取高
  • .攻击 哥布林 劣势 掷两个d20取低
  • .攻击 哥布林 命中+1d4 额外命中加值（如祝福术）
  • .攻击 哥布林 伤害+2d6 额外伤害（如偷袭、猎人印记）
  • .攻击 哥布林 附赠 标记为附赠动作
  • .攻击 哥布林 优势 伤害+1d6 叠加选项
群体攻击（一次攻击多个）：
  • .攻击 @张三 @李四 @王五
  • .攻击 哥布林1 哥布林2 哥布林3 空格隔开怪名
玩家和怪物不能混着攻击，要么全@玩家，要么全写怪名。
可能遇到的问题：
请先.装备武器先 .装备 长剑。
尚未启用模组让DM .怪物 启用 模组名。
未找到怪物让DM发 .怪物 list 确认正确名字。
③ 施法与反应
.施法 .scast
它是干嘛的：放法术。系统自动扣法术位、算命中或让目标豁免、算伤害。内置300+法术（0-9环完整法术库）。
  • .施法 火球术 哥布林
  • .施法 魔法飞弹 @张三
  • .施法 疗伤术 @李四
  • .施法 疗伤术 不写目标 = 给自己（治疗/增益类）
  • .施法 火球术 哥布林 升环1 用更高环的法术位放，伤害更高
  • .施法 灼热射线 哥布林 优势
  • .施法 火球术 哥布林 难度+2 临时提高目标豁免DC
可能遇到的问题：
法术库未找到法术名写错了，要用全名，"火球术"不能写"火球"。骰子会提示相似名字。
法术位不足.法槽 show 看剩余，.大休 恢复。
优势已忽略豁免类法术（火球术等）不掷命中骰，优势对它无效，正常提示。
.反应 · 银光锐语 .reaction
它是干嘛的：消耗1个1环法术位，给敌人的下次攻击/豁免加"劣势"，同时给友方（默认自己）加"优势"。
  • .反应 银光锐语 @敌人 @友方
  • .反应 银光锐语 哥布林 @友方 敌人是怪物时写怪名
  • .反应 银光锐语 @敌人 不写友方 = 优势给自己
.至圣斩 .smite （圣武士专用）
它是干嘛的：圣武士命中后消耗法术位，追加光耀伤害。先用 .攻击 确认命中，然后立即发本指令——两条指令配合使用，不是一条能搞定的。
怎么输入：
  • .至圣斩 哥布林 2 消耗2环法术位，对哥布林造成3d8光耀伤害
  • .至圣斩 @张三 1 对玩家至圣斩（消耗1环，2d8）
  • .至圣斩 哥布林 2 暴击 暴击攻击后使用，所有骰子翻倍
  • .至圣斩 哥布林 2 亡灵 对亡灵/魔鬼额外+1d8（规则特性）
  • .至圣斩 哥布林 3 暴击 亡灵 组合参数，顺序随便
法术位环数 → 光耀伤害骰：
1环2d8 光耀伤害
2环3d8 光耀伤害
3环4d8 光耀伤害
4环以上5d8 光耀伤害（上限）
亡灵/魔鬼在以上基础再 +1d8（暴击时同样翻倍）
标准战斗流程：
  • .攻击 哥布林 → BOT报告命中 → .至圣斩 哥布林 2
  • 暴击时：.攻击 哥布林 → BOT报告暴击 → .至圣斩 哥布林 2 暴击
⚠️ 未命中就不用发至圣斩了，系统不会自动判断你有没有命中，写了就扣法术位。
可能遇到的问题：
没有可用的X环法术位法术位用完了。.法槽 show 查剩余，.大休 恢复。
未找到怪物怪名写错，用 .怪物 list 确认名字。
伤害未结算没写目标名，指令只显示伤害但不扣血。补上怪名重发即可（但法术位已扣）。
④ HP追踪 · 死亡豁免 v2 新增
.hp 血量
它是干嘛的：查看和修改HP，带血量条和百分比显示。与海豹角色卡的 hp / hpmax 直接联动。
  • .hp 或 .血量 查看自己HP（带血量条）
  • .hp @张三
  • .hp max 45 设置最大HP
  • .hp set 30 直接把HP设为30
  • .hp heal 10 治疗10点（不超过最大值）
  • .hp dmg 15 受伤15点（有临时HP时���先扣临时HP）
  • .hp temp 8 获得8点临时HP（不叠加，取较高的）
血量条长什么样：
🟢[██████████]100% 🟡[█████░░░░░]50% 🔴[██░░░░░░░░]20% 💀[░░░░░░░░░░]0%
.deathsave / .死亡豁免
它是干嘛的：HP归零时掷死亡豁免骰（d20），系统自动记录成功/失败次数。3次成功 = 稳定，3次失败 = 角色死亡。
  • .deathsave 或 .死亡豁免 掷一次
  • .deathsave reset 重置记录（被治疗苏醒后用）
规则速查：
自然 1💀 双重失败（一次计两次失败）
2 ~ 9失败一次
10 ~ 19成功一次
自然 20🌟 立即恢复1点HP，从濒死苏醒！
3次成功角色稳定（HP仍为0，需治疗才能行动）
3次失败角色死亡
⑤ 状态效果 · 职业资源 v2 新增
.状态 / .condition
它是干嘛的：记录身上的D&D标准状态效果（目盲/中毒/麻痹等），查询时还会显示规则说明，不用翻书。
  • .状态 add 中毒
  • .状态 add 疲惫 2 带等级的状态（如疲惫1-6级）
  • .状态 list 查看当前所有状态
  • .状态 del 中毒
  • .状态 clear 清除全部状态
  • .状态 info 麻痹 查看某状态的规则说明
支持的15种标准状态：
目盲 · 魅惑 · 耳聋 · 疲惫 · 恐慌 · 擒抱 · 无行动力 · 无形 · 麻痹 · 石化 · 中毒 · 俯卧 · 束缚 · 震慑 · 昏迷
.资源 / .resource
它是干嘛的：管理职业资源（气/狂暴/神术斥责/灵感等）。.大休 和 .小休 会自动恢复对应资源。
首次设置：
  • .资源 add 气 8 short 名字、上限、恢复方式（short=短休/long=长休）
  • .资源 add 狂暴 2 long
日常使用：
  • .资源 use 气 消耗1点
  • .资源 use 气 2 消耗2点
  • .资源 list 查看所有资源（带●○点数显示）
  • .资源 restore 气 手动恢复至上限
  • .资源 set 气 3 直接设为3点
  • .资源 del 气
⑥ 增益 · 效果系统
.效果 .effect
它是干嘛的：给某人加一个临时战斗效果（buff/debuff），能模拟绝大多数"加骰子"或"给优劣势"的东西。
添加效果的完整格式：
  • .效果 添加 @张三 祝福 1d4 attack,save auto
@张三加给谁（不@就是加给自己）
祝福效果名，自己起
1d4加多少骰。纯优劣势填 0
attack,save作用范围：attack / damage / save / spellAttack / spellDamage / all，多个用英文逗号连接
auto模式：auto（自动持续）/ manual（手动触发）/ advantage / disadvantage
照抄就能用的实例：
  • .效果 添加 @张三 祝福 1d4 attack,save auto 祝福术
  • .效果 添加 @张三 猎人印记 1d6 damage auto
  • .效果 添加 @张三 精准 0 attack advantage 一次攻击优势
查看 / 删除：
  • .效果 列表   .效果 列表 @张三
  • .效果 删除 @张三 祝福
  • .效果 清除 @张三
⑦ 专长 · 激励 · 休息
.激励 .inspire （诗人专用）
  • .激励 @张三 按诗人等级自动定骰子大小
  • .激励 @张三 1d10 手动指定骰子大小
被激励的人怎么用：

.专长 .turn
  • .专长 开启 巨武器大师 近战重武器：命中-5，伤害+10
  • .专长 开启 神射手 远程武器：命中-5，伤害+10
  • .专长 开启 魔法抗性 对法术豁免有优势
  • .专长 关闭 巨武器大师
休息
  • .大休 HP回满 + 全部法术位恢复 + long类型资源恢复
  • .小休 契约法术位恢复 + short类型资源恢复
⑧ 撤回（手滑想反悔）
.撤回 .undo
它是干嘛的：撤销上一次攻击/施法/怪攻造成的所有变化——HP恢复、法术位退回、buff移除。
  • .撤回 直接输，不用加任何东西
限制：
  • 只能撤最近一次，不能连撤两次
  • 只有 45秒内 有效
  • 骰子重启后撤回记录清空

建议在私聊里录（不刷屏，更隐私）。录完群里启用就能用。
  • .怪物 哥布林 15 7 格式：名字 AC HP
给怪物加攻击动作 (.怪物 act)
  • .怪物 act 哥布林 弯刀 4 1d6+2 挥砍
哥布林给哪只怪加
弯刀动作名（自己起）
4命中加值
1d6+2伤害骰
挥砍伤害类型
加需要豁免的动作（如喷火）：
  • .怪物 act 火龙 火焰吐息 0 8d6 火焰 豁免:敏捷 难度:15 效果:灼烧
可选后缀：豁免:属性  难度:DC数值  效果:失败时的描述文字

  • .怪物 tag 稻草人 易伤 火焰 伤害翻倍

直接改怪物数值 (.怪物 set)
什么时候用：战斗中血算错了、撤回超时了、需要手动调整时。
  • .怪物 set 哥布林 hp 20 直接设为20
  • .怪物 set 哥布林 hp -5 扣5血
  • .怪物 set 哥布林 hp +8 回8血
  • .怪物 set 哥布林 ac 15
  • .怪物 set 哥布林 maxhp 30

DM 常见疑问
Q：图鉴里的怪物数据能改吗？
导入后可以用 .怪物 set 怪名 hp/ac/maxhp 数值 调整，不影响图鉴原始数据。也可以导入时直接指定HP：.怪物 库 导入 上古红龙 传奇龙 800。
Q：图鉴里没有我想要的怪物怎么办？
用手动录入：.怪物 怪名 AC HP 然后 .怪物 act 加动作，或者联系作者添加到图鉴。
Q：私聊录的怪，群里能用吗？
能。怪物按"你的账号+模组名"保存，跟在哪录无关。私聊录好，群里 .怪物 启用 模组名 调出来。
Q：换DM了怎么办？
.怪物 bind @新DM 会自动清空旧的模组启用状态，新DM重新 .怪物 启用 自己的模组即可。
Q：同一群两个DM各带一团？
轮流用。轮到A：.怪物 bind @A + A启用自己模组；换B时重新bind+启用。同一时刻只能生效一个模组。

❓ 报错自助排查（对着骰子的提示找同款）
请先 .装备 武器
为什么：没装武器就攻击了。
发 .装备 长剑。不知道有啥武器先 .武器 list。
该指令仅支持群聊环境
为什么：在私聊里用了战斗指令。
回到群里再发。
本群尚未启用模组
为什么：DM还没把怪放到本群。
让DM发 .怪物 启用 模组名。
未找到怪物【xxx】
为什么：怪名写错了或DM没录。
让DM发 .怪物 list 看正确名字，一字不差照抄。
法术位不足
为什么：这环法术位用完了。
.法槽 show 看剩余；真没了 .大休 恢复。
撤回超时了，但已经扣了法术位/HP
为什么：超过45秒无法撤回，但消耗是真实发生的。
手动补回：法术位 .法槽 调整 3 +1；玩家HP .hp heal X；怪物HP .怪物 set 怪名 hp +X（DM操作）。
法术库中未找到【xxx】
为什么：法术名写错了。
骰子会提示相似名字，对照修改。要用全名，如"火球术"不能写"火球"。
图鉴中未找到【xxx】
为什么：怪物名写错，或图鉴暂未收录该怪物。
用 .怪物 库 xxx 模糊搜索；确认没有则改用手动录入 .怪物 怪名 AC HP。
命中出现 -2 / 负数
为什么：开了巨武器大师或神射手专长（命中-5换伤害+10）。
这是正常现象。不想要就 .专长 关闭 巨武器大师。
豁免类法术"优势已忽略"
为什么：火球术这类是目标做豁免，不掷命中骰，优势对它无效。
正常提示，忽略即可。
HP显示"尚未设置HP"
为什么：没有设置 hpmax 属性。
发 .st hp=42 hpmax=42 或 .hp max 42。
死亡豁免提示"HP不为0，不需要豁免"
为什么：HP还没归零，不需要掷死亡豁免。
HP归零后（被怪物打到0血）再用 .deathsave。
大休/小休后资源没恢复
为什么：资源设的恢复类型和使用的休息不匹配（short vs long）。
.资源 list 确认恢复类型，或 .资源 restore 资源名 手动恢复。
给的优势/激励没生效
为什么：可能挂错人，或激励骰需要手动触发。
用 .效果 列表 @那个人 确认效果在不在他身上；激励骰需要被激励的人在攻击时手动写"激励"才触发。
你不是本群绑定的 DM
为什么：没有DM权限。
让群主/骰主发 .怪物 bind @你。
只有群主或骰主可以绑定 DM
为什么：绑定DM需要群主或骰主权限。
找群主帮你发 .怪物 bind @DM。

📦 属性兼容说明 v2 重要更新
v2.0.0 核心改进：插件通过海豹原生 .st 系统读取角色属性。玩家设一次 .st 力量=16 …，之后攻击/施法自动使用，属性改了也自动同步，无需重复录入。
插件读取的属性键名（按优先级）：
HPhp → HP → 生命值 → 当前生命
HP上限hpmax → HPMAX → hpMax → 生命上限 → 最大生命
ACac → AC → 护甲等级 → 护甲（默认10）
熟练加值熟练 → 熟练加值 → pb → (未设则从等级查表推算)
力量力量 → STR → str → (默认10)
敏捷/体质/…以此类推，支持中文全名 / 英文全大写 / 英文全小写
大部分DND规则模板角色卡直接兼容，无需额外配置。
🗂️ 怪物图鉴说明 v2.1.0 新增
数据来源：基于D&D 5e SRD（System Reference Document，系统参考文档），是Wizards of the Coast官方授权的公开规则集。所有数据均为官方标准，可自由使用。
自定义添加怪物到图鉴：
在脚本 MONSTER_DB 对象内追加即可，格式：
  • 必填字段：cr（挑战等级）、size（体型）、type（生物类型）、ac、hp、traits、actions
  • 可选字段：reaction（反应动作）
  • 动作字段：hit（命中加值）、dmg（伤害骰）、type（伤害类型）、extra（豁免/副作用）
具体模板见脚本文件末尾注释，或联系作者颭柳申请添加到内置图鉴。
🛠️ 可调参数（脚本顶部 CONST 区）
UNDO_TIMEOUT撤回有效时长，默认45000毫秒=45秒
MONSTER_SAVE_BASE怪物豁免加值基数，默认15（HP÷15=豁免加值，上限5）
MAX_DICE_ROLLS防刷骰上限，默认500
🧩 扩展与数据维护
扩展内容
  • 加武器：在 WEAPON_DB 里加一行，格式见脚本内注释
  • 加法术：在 SPELL_LIB 里加一条，格式见脚本内注释
  • 加图鉴怪物：在 MONSTER_DB 里追加，格式见脚本末尾模板
  • 加可计算专长：需改 cmdAttack/cmdCast 硬编码逻辑，不是纯配置
数据存储位置（海豹扩展存储）
u_UID玩家数据（武器/专长/职业等）
coll_UID_模组名怪物数据（手动录入或从图鉴导入）
effects_UID_群ID战斗效果（祝福术等）
ds_UID_群ID死亡豁免记录
cond_UID状态效果
res_UID职业资源
撤回快照是内存变量，重启即清，不影响存档。MONSTER_DB 图鉴数据内置于脚本中，不占用存储。
